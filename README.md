# Laravel 11 + Filament with Docker

Laravel 11 + Filament running in Docker with Nginx, Mysql and Redis

---
Just execute:
```bash
docker compose up -d
```

---
Access by:
```bash
http://localhost:8000
https://localhost:8001
```
